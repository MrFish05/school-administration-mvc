# School Administration System

A web-based application for managing a school, built with **PHP**, **MySQL**, and **Bootstrap** following the **MVC (Model-View-Controller)** pattern.

It allows administrators to:

- **Manage Classes** (view, add)  
- **Manage Teachers** (view, add)  
- **Manage Students** (view, add, remove)  
- **Manage Rooms** (view, add)
